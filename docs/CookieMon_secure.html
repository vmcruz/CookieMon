COOKIEMON | Documentation
CookieMon.secure
Descripción
Devuelve o asigna si las cookies serán seguras o no. Corresponde al valor opcional "secure" de una cookie.
Propiedades
Parámetros
sString Establece el valor de "secure" de la cookie.
Retorna
 String En caso de que se omita el parámetro, devuelve el valor actual de "secure"
